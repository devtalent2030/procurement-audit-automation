# Procurement Audit Automation (Project 2)
**Automated Compliance & Audit Engine for ERP procurement data**

This project demonstrates how to audit procurement exports (ERP-style Excel/CSV/JSON) using:
- **Config-driven rules (YAML)** for audit logic (change controls without code churn)
- **Rule-based validation (Pandas)** for financial/process checks
- **FOIP/PII risk scanning (NLP/NER)** to flag potential privacy leaks in unstructured notes
- **Audit-ready reports** for internal audit + compliance workflows
